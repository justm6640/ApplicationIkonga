# IKONGA Lifestyle — Monorepo

Ce dépôt regroupe les sources de l'écosystème IKONGA Lifestyle. Il démarre par un prototype d'API Node.js permettant d'explorer les principaux modules métiers décrits dans le cahier des charges.

## Contenu

| Dossier   | Description |
|-----------|-------------|
| `backend` | API Express (TypeScript) exposant les modules Nutrition, Fitness, Wellness, Beauty, Coaching et Communauté. |

## Prise en main

Consultez le [guide backend](backend/README.md) pour installer les dépendances et démarrer le serveur Express.

Les applications mobile (React Native) et web (Next.js) seront ajoutées dans des dossiers dédiés lors des itérations suivantes.
